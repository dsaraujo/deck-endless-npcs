# Deck of Endless NPCs

A Foundry VTT module that generates random NPCs based on the rules of the *[Pathfinder Deck of Endless NPCs](https://store.paizo.com/pathfinder-deck-of-endless-npcs/?searchid=322877)*. 

This module provides a simple interface within Foundry to quickly generate NPCs with details such as traits, alignment, ancestry, class, roleplay hooks, secrets, and more.

## Features

- **Random Generation:** Generates a complete NPC profile at the click of a button.
- **Actor Directory Integration:** Adds a "Generate NPC" button directly to the bottom of the Actors directory tab.
- **Foundry VTT Compatibility:** Requires Foundry VTT version 12 minimum, verified up to version 13.

## Installation

Manifest Link: https://github.com/dsaraujo/deck-endless-npcs/raw/main/module.json

### Manual Installation

1. Copy the `deck-endless-npcs` folder into your Foundry VTT `Data/modules/` directory.
2. Restart Foundry VTT if it is currently running.
3. Open your World, navigate to **Manage Modules**, and enable **Deck of Endless NPCs**.

## Usage

1. Go to the **Actors Directory** tab in Foundry VTT.
2. Look for the **Generate NPC** button at the bottom of the directory.
3. Click the button to open the generation window.
4. Click "Generate" to create your new random NPC.
